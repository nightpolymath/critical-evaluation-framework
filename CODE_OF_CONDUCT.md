# Code of Conduct

## Our Commitment

This project is committed to providing a welcoming and inclusive environment for all contributors, regardless of background, experience level, or identity.

## Standards

We expect all participants to:

- **Be respectful** — Treat others with dignity and courtesy
- **Be constructive** — Offer thoughtful critique focused on ideas, not people
- **Be honest** — Engage authentically with the framework and its limitations
- **Be collaborative** — Assume good faith and work toward shared understanding
- **Be inclusive** — Welcome diverse perspectives and experiences

## Unacceptable Behavior

The following are not tolerated:
- Harassment, discrimination, or abuse of any kind
- Trolling, bad-faith arguments, or personal attacks
- Dismissing others' expertise or perspectives without engagement
- Spam or off-topic disruption

## Reporting Issues

If you experience or witness behavior that violates this code of conduct, please report it by:
- **Emailing:** syed.shibli@daymarmetric.com
- **Subject line:** "Code of Conduct Report"

Reports will be treated seriously and with confidentiality.

---

## Scope

This code of conduct applies to all project spaces: GitHub issues, pull requests, discussions, and related communications.
